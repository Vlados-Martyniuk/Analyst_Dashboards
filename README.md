## 📂 Проєкти у цьому репозиторії

### 1. E-commerce Sales & Profitability Analysis
* **Інструменти:** Power BI, DAX, Data Modeling.
* **Суть проєкту:** Комплексний аналіз продажів (Superstore dataset) з фокусом на виявлення збиткових категорій товарів та аналіз лояльності клієнтів.
* **Головний інсайт:** Виявлено понад $156 тис. збитків у категоріях "Tables" та "Bookcases" через неоптимальну політику знижок.

### 2. Mobile Game A/B Testing (Gate 30 vs Gate 40)
* **Інструменти:** Power BI, Product Metrics (Retention).
* **Суть проєкту:** Аналіз результатів A/B тесту у мобільній грі для визначення оптимального рівня розміщення "пейволу" (gate). 
* **Головний інсайт:** Розміщення бар'єра на 30-му рівні не показує кращий Retention 1-го та 7-го дня, ніж перенесення його на 40-й рівень.
